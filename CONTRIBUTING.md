# Contributing to Telegram RPG Bot

Thank you for interest in contributing! Here's how to get started.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/telegram-rpg-bot.git`
3. Create a branch: `git checkout -b feature/amazing-feature`
4. Make your changes
5. Commit: `git commit -m 'Add amazing feature'`
6. Push: `git push origin feature/amazing-feature`
7. Open a Pull Request

## Code Style

- Follow PEP 8
- Use type hints
- Add docstrings
- Comment complex logic

## Testing

- Write tests for new features
- Run tests: `pytest`
- Ensure 100% tests pass

## Reporting Bugs

- Use GitHub Issues
- Include reproduction steps
- Include error messages/logs
- Include your environment info

## Suggesting Features

- Use GitHub Discussions
- Explain use case
- Provide examples
- Link to similar projects

## Pull Request Process

1. Update README if needed
2. Update requirements.txt if adding dependencies
3. Follow code style guidelines
4. Make sure tests pass
5. Request review from maintainers
6. Address feedback

Thank you for contributing! 🎉
